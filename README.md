## 功能

可以通过一些条件查询卡牌数据，得到想要的卡牌。目前可以通过卡牌类型、卡牌从属和关键字来查询。

前后端都是我写的，匆忙制作的CardSystem还整体上是一个ボロボロ状態，功能暂不完善，任何前后端bug或功能上的缺失、或者是卡牌数据本身的问题都可以提issue或群内爆破我。
